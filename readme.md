## Template of Vue/Express + TypeScript pnpm monorepo

### Client:

- Vue
- TypeScript
- Vite.js

### Server:

- Express
- TypeScript
- Cors, cookie-parser
- Tsup for build, tsx for dev mode

### Root scripts:

Dev mode:

```bash
pnpm dev
```

Type check:

```bash
pnpm type-check
```

Build:

```bash
pnpm build
```

Delete dist directories:

```bash
pnpm clean
```
